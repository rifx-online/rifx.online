# Rifx.online 平台折扣模型介绍文章生成提示词

## 写作风格指导

1. 叙述风格
   - 采用轻松自然的对话式语言
   - 避免过于正式或技术性的表达
   - 使用生动的比喻和例子
   - 以讲故事的方式展开内容

2. 情感连接
   - 理解读者的痛点和需求
   - 共情企业面临的困境
   - 用温暖的语气传递解决方案
   - 建立与读者的情感共鸣

3. 内容呈现
   - 避免生硬的要点罗列
   - 将数据融入故事情节
   - 用案例说明而非简单列举
   - 保持段落之间的自然过渡

4. 互动元素
   - 适时提出引导性问题
   - 设置思考点和互动环节
   - 鼓励读者参与和思考
   - 营造轻松的阅读氛围

## 文章结构设计

1. 开篇设计
   - 以企业实际困境开场
   - 引发读者共鸣和思考
   - 自然引出解决方案
   - 设置悬念和期待

2. 内容展开
   - 以问题为线索层层递进
   - 通过案例具象化优势
   - 将技术特点融入场景
   - 用故事串联各个部分

3. 过渡处理
   - 章节之间平滑过渡
   - 内容逻辑自然衔接
   - 避免生硬的段落切换
   - 保持叙述的连贯性

4. 结尾设计
   - 温暖有力的总结
   - 给出清晰的行动建议
   - 展望美好的未来
   - 留下深刻的印象

## 重点内容要求

1. 成本优势展示
   - 通过具体案例说明
   - 结合企业实际场景
   - 用故事形式呈现数据
   - 突出实际价值

2. 技术特点介绍
   - 将技术融入应用场景
   - 用类比方式解释原理
   - 避免晦涩的技术术语
   - 强调实用价值

3. 应用案例分享
   - 讲述真实的企业故事
   - 展示转变的过程
   - 分享实际的收益
   - 突出可借鉴性

4. 使用指南说明
   - 以教学的方式展开
   - 设置互动和思考点
   - 强调操作的简便性
   - 突出使用体验

## 语言风格规范

1. 表达方式
   - 使用生动活泼的语言
   - 避免生硬的专业术语
   - 适当使用比喻和类比
   - 保持亲切自然的语气

2. 句式选择
   - 避免过长的复杂句
   - 使用简洁明了的表达
   - 适当使用修辞手法
   - 注重语言的节奏感

3. 情感基调
   - 保持积极乐观的态度
   - 传递温暖和关怀
   - 建立信任和共鸣
   - 激发行动的动力

4. 互动设计
   - 设置思考性问题
   - 引导读者参与讨论
   - 预设读者的疑问
   - 及时给出解答

## 质量检查要点

1. 内容检查
   - 故事性是否充足
   - 案例是否生动
   - 数据是否具象
   - 逻辑是否连贯

2. 语言检查
   - 是否亲切自然
   - 是否易于理解
   - 是否有感染力
   - 是否有吸引力

3. 结构检查
   - 过渡是否自然
   - 层次是否分明
   - 重点是否突出
   - 节奏是否适当

4. 效果检查
   - 是否引发共鸣
   - 是否解决困惑
   - 是否激发行动
   - 是否留下印象

## 预期目标

创作一篇温暖而专业的文章，通过生动的叙述和真实的案例，帮助企业了解和选择Rifx.online的折扣模型服务。文章应该：

1. 让读者感受到被理解和关心
2. 通过故事传递专业信息
3. 用案例说明实际价值
4. 激发读者的行动意愿

## 输出流程

### 第一步：输出文章大纲
1. 基于写作要求输出详细大纲
   - 完整的章节结构
   - 每章主要内容要点
   - 重点案例规划
   - 过渡段落设计

2. 与用户确认大纲
   - 征求用户意见
   - 根据反馈调整
   - 确定最终框架
   - 明确重点方向

### 第二步：分章节输出
1. 按用户指引的顺序输出
   - 一次输出一个章节
   - 保持章节完整性
   - 确保前后连贯
   - 突出重点内容

2. 每章节输出后等待确认
   - 请用户审阅内容
   - 接受修改建议
   - 进行必要调整
   - 确认后继续下章

### 第三步：整体优化
1. 完成所有章节后
   - 检查整体连贯性
   - 优化过渡段落
   - 统一写作风格
   - 完善细节描述

2. 最终确认
   - 整体效果确认
   - 统一格式规范
   - 检查质量要点
   - 完成终稿输出
